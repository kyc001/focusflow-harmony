# PRD: 以毕业论文标准重写论文

## 背景

"知序"（ZhiXu）是一个面向 HarmonyOS 的本地优先学习管理应用，作为鸿蒙应用开发课程的期末大作业。现有论文（`course-report.tex`）内容过于单薄，仅约 176 行、5 章，远未达到南开大学本科毕业论文 8000 字正文的要求。需要以毕业论文标准完全重写。

## 参考依据

- **南开大学 2020 年本科毕业论文指导手册**：正文≥8000 字，结构含封面、声明、中英文摘要、目录、正文、附录、参考文献、致谢
- **论文样张**：8 章结构，含引言、方法论、数据分析、结论，有公式、表格、图表编号
- **现有 LaTeX 模板**：NKU-thesis-template-2020，book 类，XeLaTeX 编译

## 需求

### 功能需求

1. **完全重写 `course-report.tex`**：从当前 5 章扩展为 7-8 章，正文 ≥8000 中文字符
2. **新增"相关工作与技术"章节**：文献综述，引用 HarmonyOS、ArkTS、移动学习、AI 辅助学习、Pomodoro 技术等方面的真实文献
3. **新增"系统测试与验证"章节**：功能测试、构建验证、性能观察
4. **重写摘要**：中英文摘要需要更完整、更学术化，关键词更精准
5. **完善参考文献**：替换 `nkthesis.bib` 中的模板样例，添加与论文主题相关的真实参考文献（≥15 篇），正文中使用 `\cite{}` 引用
6. **添加图表**：至少包含架构图、数据流图、ER 图、页面功能表、数据库表结构表、AI 模块流程图等
7. **添加算法描述**：AI 本地提炼算法、任务生成算法的形式化伪代码
8. **完善致谢**：替换占位符，写入真实的课程致谢内容
9. **更新 main.tex**：包含 acknowledgements、references（移除 manual.tex 引用）
10. **更新 resume.tex**：替换占位符为真实个人信息（或移除）

### 格式要求（来自指导手册）

| 元素 | 格式 |
|------|------|
| 正文 | 小四号宋体（12pt），1.5 倍行距 |
| 一级标题 | 小三号黑体（15pt） |
| 二级标题 | 四号黑体（14pt） |
| 三级标题 | 小四号黑体（12pt） |
| 摘要标题 | 四号黑体，居中 |
| 关键词 | 小四号黑体，顶格 |
| 参考文献 | 五号宋体（10.5pt），[1] 编号 |
| 表格 | 三线表，表题在表格上方居中 |
| 图片 | 图题在图片下方居中 |
| 公式 | 居中，编号右对齐 |
| 页边距 | A4，默认（上下 2.54cm，左右 3.17cm） |

### 验收标准

1. 正文 ≥8000 中文字符（不含代码和参考文献）
2. 至少 7 章结构完整
3. 正文中至少 15 处 `\cite{}` 引用
4. `nkthesis.bib` 中至少 15 条与主题相关的真实文献条目
5. 至少 6 幅图（架构图、流程图、ER 图、截图等）
6. 至少 5 个表（功能对比、数据库结构、技术选型等）
7. 至少 2 个算法伪代码
8. 中英文摘要各 ≥300 字
9. 致谢内容真实完整
10. `main.tex` 正确引用所有必要文件
11. 使用 `xelatex` + `biber` 编译通过（无报错）

## 约束

- 保持现有 LaTeX 模板结构不变（NKThesis.sty 不修改）
- 论文主体用中文撰写，英文摘要同步更新
- 不展示大段源码（指导手册要求第三章和第四章不贴代码），用文字描述和图表代替
- 参考文献使用 BibLaTeX 格式（`nkthesis.bib`）
