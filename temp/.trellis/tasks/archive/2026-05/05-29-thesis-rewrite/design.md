# Design: 论文重写技术设计

## 论文结构设计

基于南开大学毕业论文样张（8 章）和指导手册要求，重写后的论文结构如下：

### 目标章节结构

| 章 | 标题 | 内容概述 | 预估字数 |
|----|------|----------|----------|
| 摘要 | 摘要 / Abstract | 中英文摘要 + 关键词 | 各 300+ 字 |
| 1 | 引言 | 背景、问题、研究目标、论文组织 | 1200 字 |
| 2 | 相关工作与技术基础 | HarmonyOS 生态、ArkTS/ArkUI、移动学习、Pomodoro、AI 辅助学习 | 1500 字 |
| 3 | 系统需求分析 | 需求分析、用例设计、非功能需求 | 1200 字 |
| 4 | 系统设计 | 总体架构、模块设计、数据库设计、AI 模块设计、接口设计 | 2000 字 |
| 5 | 系统实现 | 各模块实现描述（不贴代码）、关键技术问题与解决方案 | 1800 字 |
| 6 | 系统测试与验证 | 测试策略、功能测试、构建验证、兼容性 | 1000 字 |
| 7 | 总结与展望 | 工作总结、创新点、不足与展望 | 800 字 |
| 参考文献 | 参考文献 | ≥15 条真实文献 | — |
| 致谢 | 致谢 | 课程致谢 | 200 字 |

**正文总计**：约 9500+ 字，满足 ≥8000 字要求。

### 各章详细设计

#### 第一章 引言（~1200 字）

- 1.1 研究背景：移动学习碎片化问题、HarmonyOS 生态发展、AI 赋能学习工具的趋势
- 1.2 研究目标：设计本地优先的学习管理应用，整合资料管理、任务分解、番茄专注、复盘统计
- 1.3 研究内容：5 条编号列出核心工作
- 1.4 论文组织：逐章说明论文结构

#### 第二章 相关工作与技术基础（~1500 字）

- 2.1 HarmonyOS 与 ArkTS：介绍 HarmonyOS 分布式架构、ArkTS 语言特点、ArkUI 声明式 UI
- 2.2 移动学习管理工具现状：对比 Notion、Forest、滴答清单等现有工具
- 2.3 番茄工作法：Pomodoro 技术原理及其在学习工具中的应用
- 2.4 AI 辅助学习：LLM 在学习助手、摘要生成、任务推荐中的应用
- 2.5 本章小结

> 每个技术点引用 2-3 篇文献

#### 第三章 系统需求分析（~1200 字）

- 3.1 系统概述：产品定位、目标用户、使用场景
- 3.2 功能需求：5 个核心页面的功能描述（用例列表）
- 3.3 非功能需求：离线可用性、性能、安全性、可扩展性
- 3.4 本章小结

#### 第四章 系统设计（~2000 字）

- 4.1 总体架构设计：本地优先架构、MVVM 分层（**含架构图**）
- 4.2 数据流设计：UI → Store → RDB → 刷新 → UI（**含数据流图**）
- 4.3 数据库设计：4 张表的字段定义（**含 ER 图 + 表结构表**）
- 4.4 AI 模块设计：本地规则 + 远程模型双模式（**含流程图 + 算法伪代码**）
- 4.5 后端接口设计：RESTful API 设计（**含接口表**）
- 4.6 本章小结

#### 第五章 系统实现（~1800 字）

- 5.1 开发环境与工具：DevEco Studio、hvigor、Spring Boot
- 5.2 今日页实现：进度展示、快速添加、AI 建议
- 5.3 任务管理实现：搜索、筛选、四象限优先级、软删除
- 5.4 学习资料库实现：多类型录入、AI 提炼、一键生成任务
- 5.5 番茄专注实现：计时器、后台守护、独立窗口
- 5.6 复盘统计实现：本地统计、ArkWeb 图表、学习森林
- 5.7 关键技术问题与解决方案
- 5.8 本章小结

#### 第六章 系统测试与验证（~1000 字）

- 6.1 测试策略：功能测试、构建验证、兼容性测试
- 6.2 功能测试：各页面功能测试结果（**含测试用例表**）
- 6.3 构建验证：HAP 构建、JAR 构建结果
- 6.4 性能与兼容性观察
- 6.5 本章小结

#### 第七章 总结与展望（~800 字）

- 7.1 工作总结：完成了什么、达到了什么效果
- 7.2 主要创新点：本地优先 + AI 增强 + 学习闭环
- 7.3 不足与展望：OCR、同步 UI、AI 增强、无障碍

## 编译兼容性

- 使用现有 `NKThesis.sty`，不修改
- 保持 `main.tex` 基本结构，仅调整 include 顺序和内容
- 新增的图需要放在 `figures/` 目录下
- 参考文献使用 BibLaTeX + Biber 编译链
